Software

There are a number of software tools available for working with ZX-diagrams, ranging from interactive graphical editors to programmatic libraries for automated simplification and circuit optimisation.

ZX Sketch

ZX Sketch is a free, browser-based ZX-calculus diagram editor. It provides an interactive canvas for drawing ZX-diagrams, applying rewrite rules, and constructing step-by-step proofs â€” all directly in your browser with no installation required.

Features include 19 built-in rewrite rules, 18 simplification strategies, a proof construction mode, and export to TikZ, SVG, PNG, and OpenQASM formats.

Try ZX Sketch in your browser  |  Learn more  |  GitHub

PyZX

PyZX is a Python library for automatically simplifying ZX-diagrams and optimising quantum circuits, written by a team at Radboud University. It implements state-of-the-art T-count reduction and circuit optimisation procedures.

You can try PyZX directly in your browser using the in-browser demo, which demonstrates the T-count optimising procedure presented in this paper.

GitHub  |  In-browser demo

TikZiT

TikZiT is a graphical user interface for creating TikZ diagrams. Many papers that use ZX-diagrams use TikZiT to produce publication-quality figures.
