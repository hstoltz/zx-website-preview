ZX Sketch

ZX Sketch is a free, browser-based ZX-calculus diagram editor. It provides an interactive canvas for constructing and manipulating ZX-diagrams directly in your browser, with no installation or account required.

The idea is to make something like Desmos but for ZX-calculus: an interactive and accessible tool where anyone can manipulate ZX-diagrams directly in the browser.

Features

  • Interactive drag-and-drop diagram editor with Z spiders, X spiders, H-boxes, and wires
  • 19 built-in rewrite rules (spider fusion, bialgebra, Hopf, colour change, pi-copy, and more)
  • 18 simplification strategies (including full Clifford simplification)
  • Proof construction mode for building step-by-step derivations
  • Export to TikZ, SVG, PNG, and OpenQASM formats
  • Every rule uses custom match-discovery logic and precondition checks for correctness
  • Uses PyZX (v0.9.0) running in the browser via Pyodide for simplification
